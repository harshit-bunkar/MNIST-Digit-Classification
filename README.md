## 📌 Project Overview

To build a Neural Network model that recognizes handwritten digits (0-9) from 28x28 pixel grayscale images.

## 🎯 Objectives

- Analyze and understand the dataset
- Clean and preprocess the data
- Build a model using **Artificial Neural Network (ANN) built with TensorFlow & Keras**
- Train the model and make predictions
- Evaluate the model's performance

## 📊 Dataset Information

The MNIST dataset, loaded directly from Keras. It contains 70,000 grayscale images of handwritten digits (60,000 for training and 10,000 for testing), each 28x28 pixels.

**Key features:**

- 28 x 28 = 784 pixel values per image
- Pixel intensity range 0-255 (normalized to 0-1)

**Target: digit label from 0 to 9 (10 classes)**

|--------|-------|

The trained neural network achieved approximately 96.7% accuracy on the 10,000-image test set. The model was also tested on a custom handwritten-digit image and predicted the digit correctly.
